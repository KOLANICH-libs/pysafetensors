A lib parsing [`safetensors` format](https://github.com/huggingface/safetensors) without involving Rust.

This parser relies on the [Kaitai Struct spec](https://github.com/KOLANICH-specs/kaitai_struct_formats/blob/safetensors/serialization/safetensors.ksy).

Currently undocumented, see the source code.
